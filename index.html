Generate a one-click link that installs your mod through TriOS

Make a one-click install button for your mod's forum post. Fill in the fields below, then copy the code into your post — anyone with TriOS can install your mod (and its dependencies) in one click.

  1. Your mod — paste the link to your mod's .version file (recommended: it auto-updates every release). No .version file? Paste a direct link to your mod's .zip download instead.
  2. Mod ID (optional) — the id value from your mod_info.json. Lets TriOS skip the mod if it's already installed.
  3. Dependencies (optional) — click Add dependency and paste each required mod's .version or .zip link. Each one can have its own ID.
  4. Copy & paste — leave the format on BBCode badge, click Copy, and paste it into your forum post. That's it.

lightbulb Prefer a .version file URL over a direct download — it self-updates with each release.
